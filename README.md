# BabyGumi | Landing Page Interactiva

### Descripción
Página web para un emprendimiento artesanal de amigurumis y accesorios para bebés en Tucumán. Permite a los usuarios explorar un catálogo, gestionar un carrito de compras y finalizar el pedido vía WhatsApp.

### Tecnologías Utilizadas
* HTML5
* CSS3 (Diseño responsivo y variables)
* JavaScript (Manipulación del DOM y Eventos)

### Funcionalidades de Interacción (DOM)
1. **Modo Noche:** Cambio dinámico de paleta de colores para mejorar la accesibilidad.
2. **Sistema de Carrito:** Gestión dinámica de productos (agregar/eliminar) con cálculo automático de descuentos.
3. **Integración con WhatsApp:** Generación automática de mensajes basados en la selección del usuario.

### Instrucciones para ejecutarlo
1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador moderno.
